# 使用範例

## 基本流程

### 1. 連接到 BBS
```
請使用 bbs_connect 工具連接到巴哈姆特 BBS
```

### 2. 查看歡迎畫面
```
請使用 bbs_get_screen 工具查看目前畫面
```

### 3. 登入流程

#### 輸入帳號
```
請使用 bbs_send 工具，輸入我的帳號：myusername
```

#### 按下 Enter
```
請使用 bbs_send_key 工具，按下 enter 鍵
```

#### 輸入密碼（不建議）
```
不建議使用 bbs_send 直接輸入密碼，避免密碼出現在對話或操作紀錄中
```

#### 建議做法：使用自動登入
```
請先在 .env 設定 BBS_USERNAME 與 BBS_PASSWORD
請使用 bbs_auto_login 工具自動登入
```

#### 再次按下 Enter
```
請使用 bbs_send_key 工具，按下 enter 鍵
```

### 4. 瀏覽看板

#### 使用方向鍵移動
```
請使用 bbs_send_key 工具，按下 down 鍵
請使用 bbs_send_key 工具，按下 up 鍵
```

#### 選擇看板
```
請使用 bbs_send_key 工具，按下 enter 鍵進入看板
```

### 5. 搜尋看板
```
請使用 bbs_send 工具，輸入 s（搜尋指令）
然後輸入看板名稱
```

### 6. 中斷連線
```
請使用 bbs_disconnect 工具
```

## 常用 BBS 指令

- `s` - 搜尋看板
- `q` - 離開/返回上一層
- `→` - 閱讀文章
- `←` - 返回
- `↑` - 上移
- `↓` - 下移
- `Home` - 跳到第一項
- `End` - 跳到最後一項
- `r` - 回覆文章（需登入）
- `w` - 發表文章（需登入）

## 進階使用

### 自動化登入
你可以將多個指令串在一起：

1. 連接
2. 查看畫面
3. 使用 `bbs_auto_login`
4. 查看登入後畫面

### 讀取特定看板文章

1. 登入後
2. 搜尋看板（按 s，輸入看板名稱）
3. 進入看板（按 Enter）
4. 瀏覽文章（上下鍵）
5. 閱讀文章（按 Enter 或右鍵）

## 注意事項

1. **編碼問題**：BBS 使用 Big5 編碼，某些特殊字元可能顯示不正確
2. **ANSI 控制碼**：畫面可能包含 ANSI 顏色控制碼，需要適當處理
3. **延遲**：每次操作後會自動等待畫面更新與穩定，實際延遲依操作而定
4. **連線穩定性**：長時間未操作可能被斷線
5. **畫面更新**：有些操作需要等待較長時間才會有回應

## 除錯

如果遇到問題：

1. 檢查是否已連接：使用 `bbs_get_screen` 查看狀態
2. 重新連接：先 `bbs_disconnect`，再 `bbs_connect`
3. 查看錯誤訊息：錯誤訊息會在回應中顯示
4. 檢查網路：確保可以連接到 `wss://term.gamer.com.tw/bbs`
